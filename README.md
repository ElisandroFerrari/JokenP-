<h1>JokenPô - Game Edition </h1>
<br>
Este é um projeto de jogo clássico de Pedra, Papel e Tesoura desenvolvido durante a trilha de JavaScript do DevClub.
O foco principal foi o aprendizado de funções, lógica de programação, manipulação do DOM e design responsivo com Media Queries.
<br>
<br>

Funcionalidades
Lógica de Jogo: O usuário joga contra a CPU, que possui uma escolha aleatória via Math.random().

Placar em Tempo Real: Atualização dinâmica da pontuação do jogador e da máquina.

Design Responsivo: Ajustado para funcionar perfeitamente em modo retrato (portrait) e paisagem (landscape) em dispositivos móveis.

Identidade Visual: Uso da fonte Diplomata e cores integradas ao tema de moldura antiga.
